About the Website

This an open source website using firebase as a backend to store all the data. We have collected the data on our own and the goal is to help find monitor prices easily. If you would like to contribute to the website please visit our github: Link to Github

Note: This website does not store any sort of IP's or track location or anything else. This is just simply built to help you find monitors easily

If you believe that some of the prices or information is wrong, please visit github to send a pr to update it.
